# PaddleOCR-VL-1.5 微调最佳实践

本目录提供了基于 **PaddleOCR-VL-1.5** 模型进行微调的最佳实践教程。

## 模型简介

PaddleOCR-VL-1.5 是 PaddleOCR-VL 的全新升级版本（0.9B 参数），在文档解析领域（OmniDocBench v1.5）取得了 SOTA 性能。相较于前代，它在**抗干扰鲁棒性**、**多任务扩展（印章/文本定位）**以及**长文档与多语言支持**方面实现了显著突破。

## 微调任务教程

本目录下包含两个典型应用场景的详细微调指南，请根据您的需求阅读相应文档：

### 1. 区域识别 (RegionOCR)
*   **任务目标**：给定图像及目标区域（Region of Interest, ROI），精准识别该区域内的文字内容。
*   **应用场景**：票据/表单字段提取（如金额、单号）、复杂版面指定区域解析、印章与签名提取等。
*   **文档链接**：👉 [点击阅读 RegionOCR 微调教程](./region_ocr.md)

### 2. 复杂表格识别 (Complex Table Recognition)
*   **任务目标**：精准理解并还原复杂表格的结构与内容。
*   **应用场景**：处理多重合并单元格、嵌套表头、无线框表格、跨页表格等复杂结构。
*   **文档链接**：👉 [点击阅读 Complex Table Recognition 微调教程](./table_ocr.md)
